# 四个止盈策略

## 策略一：通过仓位管理被动止盈
1. 持仓标的控制在5-8个：这个数量是散户平衡“分散风险”和“精力聚焦”的最优解。太少，容易踩雷；太多，研究不透。5-8个，既能覆盖不同行业，又能对每个标的保持深度跟踪。
2. 单个标的仓位在5%-25%，超过上限就减仓：这是最核心的“自动止盈”触发器。当一只股票因为上涨，仓位被动超过25%时，需要无条件减仓，把仓位拉回上限以内。这背后的逻辑是：任何单一资产都不应过度影响整体组合。减仓不是不看好，而是为了锁定利润、控制风险，同时减仓的钱可以去买入其它风险更小的股票。
3. 随时保留最少10%的现金仓位：这笔钱是“战略预备队”，这种做法有几大好处：1.市场暴跌时有钱加仓，心态会稳很多；2.发现好的机会时能果断出击；3.保持研究新机会的动力，而不是满仓躺平只顾着幻想暴涨。

这套规则的本质，是用纪律对抗人性。它不保证我卖在最高点，但能确保我在上涨过程中不断“落袋为安”，将浮盈转化为实实在在的利润，从而有效控制整体账户的盈利回撤。投资是场马拉松，活得久比跑得快更重要。这套仓位管理法，就像是投资路上的“安全带”。

## 策略二：通过顶部指标主动止盈
1. 股价进入历史高位区域：当一只股票的股价逼近或创下历史新高，或者处于其上市以来股价的前10%分位数内时，需要立刻进入高度警惕状态。历史高位意味着上方已无套牢盘，但也意味着所有持仓者都是获利盘，任何风吹草动都可能引发集中抛售。这个位置，风险的收益比往往不再具有吸引力。
2. 成交量/换手率异常暴增：这是最可靠的短期见顶信号。当股价在高位运行时，如果单日成交量或换手率突然放大到平时水平的2-3倍以上，形成“天量”，这通常是主力出货、多空分歧急剧加大的明确标志。所谓“天量见天价”，巨大的成交量背后，是获利盘的集中了结，上涨动能很可能在此刻衰竭。
3. 技术指标的顶背离：当股价创出新高，但诸如RSI（相对强弱指标）、MACD等技术指标却未能同步创出新高，反而走低，形成“顶背离”。这表明股价的上涨已经缺乏内在动能的支撑，是上涨趋势可能逆转的先行信号。

结合上面三条来看2026年1月底的紫金股价，可以说还是挺准确的。遇到这种情况，正确的做法应该是在1月26号成交量暴涨时就开始撤退，后面就是越涨越减，随时准备清仓。回过头来看当然能很清晰地看出那里是近期的顶部，但在当时那个大家都很狂热的氛围里如何意识到顶部将至？这是我想构建风控管理系统的一个主要原因，做法就是让系统监控各项指标规则，并每天给我发送各种提醒，具体如何实现会在后续文章中讲述。

## 策略三：移动止盈
1. 成本线保护法（新手最稳）
  - 规则： 买入后只要股价不跌破买入成本价就坚定持有。
  - 移动： 当股价上涨了20%后，把止盈线移动到买入成本价 + 5% 的位置。如果股价回调跌破这个位置就卖出一半；如果继续跌就全卖。这里的具体数值可以根据个人喜好或所持股的股性来动态设定。
  - 目的： 确保这笔交易绝不亏损，哪怕后面跌回原点，也能保本或微利退出。

2. 关键均线防守（趋势跟踪）
  - 规则： 重点看 20日均线 和 60日均线。激进版： 只要股价不破 20日均线 就一直拿着。一旦有效跌破（比如连续3天或跌幅超过3%），无条件减仓一半或清仓。稳健版： 以 60日均线 作为最后防线。只要股价在60日均线上方，就认为大趋势还在，一旦跌破立即清仓。

3. 最高点回撤法（利润回撤控制）
  - 规则： 盯着这只股票的近期最高价（比如过去3个月的最高价）。
  - 移动： 设定一个回撤比例，比如 15%。如果股价从100元跌到85元，我就卖出。随着股价新高，这个止盈线也跟着上移。如果涨到120元，我的止盈线就自动变成102元。
  - 目的： 确保利润不会回撤超过设定的比例，强制“落袋为安”。

## 策略四：突发利空时无条件止盈
很多时候亏大钱，不是因为技术不好，而是因为“心存侥幸”。当持仓股突发重大利空时，第一反应往往是：“影响大不大？是不是错杀？要不要等反弹再卖？”等等。前段时间中矿突发利空时，我就是这种心理，由于持有了一年多时间，加上有200%的盈利，就没把津巴布韦的禁运当回事儿，现在想想确实不应该，正确的做法应该是不管怎样先减半仓再说。

# 为什么难以做到高位止盈？
想要做到高位止盈，关键要克服以下几点：
1. 害怕卖飞：在一段时间的震荡上涨中，卖飞几次后产生强烈的懊悔情绪，导致在后续的上涨中越来越不敢卖出，哪怕是在明显的高位，也会想着还能再创新高。再加上有了一定的利润-垫，风偏不自觉升高，于是盲目看多，风险意识全无。木匠叔曾多次强调不要有“抄底/逃顶/套牢/卖飞”这类想法，不过想做到这一点还是非常困难的。
2. 锚定效应：当股价达到区间新高后回调时，会自然而然地盯着前高的价格，想着现在卖就亏了，起码要等到涨回前高再卖。实际上就算涨回前高也大概率不会卖，但如果继续下跌会更难下定决心卖出，直到跌回成本线或心理产生恐慌时，才会匆匆忙忙慌不择路地卖出，于是造就了“保本出”这个经典策略。
3. 回调加仓：比锚定效应更可怕的是在回调时加仓，由于眼睛只顾盯着前高，加上利润垫带来的风偏升高，会忍不住想在新高后的大跌中加仓，陷入一种“只要稍微一反弹我就能盈利更多”的赌徒心态。这种操作如果是在股价上行期发生，并成功博到了几次反弹，那就更容易在后续回调时引起加仓的冲动。这是一种很危险的操作，就像财主置顶帖所说的“最容易虧錢的方法是趁回吐買入，比追高要危險十倍百倍，切記！”
4. 信息茧房和集体幻想：当长时间拿着一只股票，且这只股票又带来了很可观的利润时，会不自觉地跟这只股票产生感情，看着它哪儿哪儿都是好的，似乎连公司管理层都看起来特别地亲切，这会导致持有者慢慢失去基本的判断力，并陷入一种越来越深的幻觉。特别是当大量散户同时陷入这种幻觉时，大家会变得只关注利好消息，遇到利空也不太重视，遇到回调就相互抱团打气，最终进入“未来一片美好，人生即将迎来暴富”的集体幻想中。一旦进入这种状态，就好像被洗脑了一样，哪怕账户受到沉重打击也不愿意睁开眼睛面对现实，而是转移视线，将怨气发泄在其他的人和物上面。

# 结语
股票的迷惑性，不在于其内在价值，而在于它给予交易者的无限幻想。我们最大的敌人，从来不是市场，而是自己脑海中那些非理性的情绪与不切实际的期待。唯有以清晰的逻辑取代盲目的期盼，用明确的规则驯服冲动的本能，我们才能摆脱被市场和情绪左右的命运，蜕变为一名真正有纪律、有边界的“投资者”。
