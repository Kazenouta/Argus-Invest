## 投资哲学框架

### 核心理念
1. **风险第一**：所有收益都来自风险本身，风险是投资里最值钱的东西
2. **长周期思维**：坚信主逻辑和长逻辑，不为短期波动所动
3. **体系化思考**：强调完整阅读、系统分析，反对碎片化信息
4. **长期主义**：相信市场永远有机会，持续研究才有意义

### 仓位管理原则
- 反对过度集中（单吊梭哈）和过度分散（开超市）
- 建议持有 **3-5 支票**（研究深入的投资者可 5-8 支）
- 单只个股仓位不超过 20%，极端情况不超过 25%
- 始终保持 **最少 10% 现金比例**
- 持仓之间要有跨周期关系或对冲关系
- 大账户（80-90%）+ 小账户（10-20%，打野/衍生品）

### 技术分析观
- 技术分析**有用**，但单独用没效果
- 技术分析的有效性建立在对基本面认知有效的前提下
- 技术分析最大用处是形成**交易纪律**，强制止盈/止损
- 见顶信号：换手率和成交量暴增（尤其是 4-5 倍），是综合判断结果
- 技术指标是**深度投研的结果，而不是原因**

### 市场观
- 股市是**货币现象**，是整个市场货币形势的反应
- A股长牛的基础：资金跟着产业走，不是跟着小作文
- 产业链轮动：原材料→能源→化工制造→日常消费→电子信息→物流→房地产金融→高端消费科技
- 做投资要**理解市场、尊重市场、敬畏市场**

## 核心资产观点

### 黄金
- 超级大牛市未结束，货币信用下降是核心驱动力（不是战争）
- 金价上涨 = 美元/全球主权货币信用和购买力下降
- 只要主权货币信用继续下降，黄金就会继续涨
- A股黄金股定价机制畸形，紫金矿业是相对最靠谱的黄金股
- **当前（2026-04）**：等待，等中美会谈 + 战争走势明朗；加息概率增加，不着急布局

### 原油/能源
- 霍尔木兹海峡是全球经济的咽喉，封锁导致能源危机持续
- 中东战争持续 = 原油持续上涨，直到爆发大危机导致需求大衰减
- 特朗普目的是获取能源控制权，不会轻易撤兵（政治风险太大）
- 美军正在大规模军事部署，战争不确定性极大
- **当前**：能源是最高确信度板块，加仓旧能源（嗨油、中妹煤炭ETF）

### 铜
- 长期逻辑：供给侧实质缺口，几年内无法解决
- 螺旋型上升走势，只要全球经济不崩盘就持续
- **当前（2026-04）**：受战争负面影响，担忧流动性/衰退，走势不如铝，谨慎等待

### 电解铝
- 中国电解铝优势极强，无法被替代（资源+能源+技术壁垒）
- 铝的弹性高于铜，长期胜率铜更高
- **当前（2026-04）**：等待局势明朗，保持仓位

### 化工
- 长期逻辑不变，有原材料和技术优势的企业依旧看好
- 保供操作对龙头企业中长期有利无害
- **当前（2026-04）**：长期持有，等局势明朗，不着急交易

### 大盘（A 股）
- 2024年7月写《老A和他的长牛》，924行情启动验证了判断
- 牛市建立：货币+产业面+资金面三周期共振
- **当前（2026-04）**：风险大于机会，融资余额2.6万亿高位，科技板块泡沫未充分释放，成交额跌破1.6万亿，考验3800支撑，可能回调至3700/3600
